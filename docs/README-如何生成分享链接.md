# 如何生成可分享链接

这个文件夹里的网页目前是本地网页，只有你的电脑能直接打开。要发给朋友，需要先把整个文件夹上传到网页托管平台，平台会给你一个可复制的链接。

## 最简单方法：Netlify Drop

1. 打开浏览器访问：`https://app.netlify.com/drop`
2. 把整个文件夹 `马梦雨-古筝商演个人IP展示` 拖进去，或者拖入压缩包。
3. 等待上传完成后，页面会自动生成一个网址。
4. 把这个网址发给朋友即可。

建议分享文案：

> 你好，这是我的古筝商演个人展示页，里面包含演出视频、曲库、服装造型和联系方式，方便您快速了解演出风格与可合作场景。

## 上传前请先检查

- 首页是否能打开；
- “曲目单 / 曲库”页面是否能打开；
- 视频是否能播放；
- 联系方式是否已经改成真实微信、电话或二维码；
- 如果暂时不想公开电话，可以只放微信二维码。

## 如果上传后视频无法播放

通常是视频文件太大或格式兼容问题。建议：

- 视频优先使用 `.mp4`；
- 单个视频尽量压缩到 50MB 以内；
- 首页只放 1-3 个精选视频，其余可以后续放到视频号、抖音、小红书，再在网页里放链接。
