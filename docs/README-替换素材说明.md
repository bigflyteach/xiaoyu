# 马梦雨古筝商演展示包

双击 `打开网站.bat` 即可预览。也可以直接双击 `index.html`。

## 已完成的页面结构

- 首屏个人 IP 定位
- 精选演出视频与更多现场视频
- 独立曲库页面（38 首真实曲目，支持分类与搜索）
- 婚礼、宴会、品牌活动、文化空间四类合作场景
- 服装造型选择
- 合作流程
- 个人介绍
- 联系与档期咨询侧边栏
- 手机、平板、电脑自适应

## 发布前必须修改的联系方式

用记事本打开 `index.html`，搜索下面三段文字并替换：

1. `请填写您的微信号`
2. `请填写您的手机号`
3. `微信二维码待替换`

微信二维码图片建议放入：

`assets/images/05-联系方式/wechat-qr.png`

然后把 `index.html` 里的 `<div class="qr-placeholder">...</div>` 替换为：

```html
<img class="qr-placeholder" src="assets/images/05-联系方式/wechat-qr.png" alt="马梦雨微信二维码">
```

同时把复制按钮的 `data-copy` 改成真实微信号。

## 图片分类规则

- `01-首屏主视觉`：横版或可裁切的大图，人物与古筝关系清晰
- `02-商演现场`：真实婚礼、宴会、商场、品牌活动照片
- `03-服装造型`：每套服装一张全身照，建议统一背景
- `04-个人形象`：用于个人介绍，清晰、自然、有亲和力
- `05-联系方式`：微信二维码、公众号二维码等
- `06-品牌合作`：后续可放品牌 Logo 或合作方授权照片

替换图片时保持原文件名不变，网页会自动更新，不需要改代码。

## 视频分类规则

- `01-精选演出/featured.mp4`：第一条主推视频，建议 30–90 秒
- `02-更多现场/live-02.mp4`
- `02-更多现场/live-03.mp4`
- `02-更多现场/live-04.mp4`

网页端推荐 MP4（H.264 编码）。若视频加载慢，建议导出为 1080p、码率 5–8 Mbps。

## 发给活动方之前

- 填好微信号、手机号和二维码
- 把视频标题改成真实活动或曲目名称
- 检查每张照片和视频是否已获得公开展示许可
- 用手机打开一次，确认视频声音和二维码扫码正常
- 正式对外建议部署为网址，不要直接发送整个大文件夹

当前页面不展示价格，档期、时长与合作细节通过联系方式单独沟通。
